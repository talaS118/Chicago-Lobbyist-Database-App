# Chicago Lobbyist Database Application

This project is a Python and SQLite application that interacts with a database containing information about lobbyists, employers, clients, and compensation data.

The application allows users to query and update lobbyist data through a simple command-line interface.

## What This Project Does
- Connects to a SQLite database using Python
- Executes SQL queries to retrieve and update data
- Displays results in a user-friendly, menu-based interface

## Features
- Search lobbyists by first or last name
- View detailed information for a specific lobbyist
- Find the top N lobbyists by total compensation for a given year
- Add a registration year for a lobbyist
- Update a lobbyist’s salutation

## Technologies Used
- Python
- SQLite
- SQL

## Files
- `main.py` – Handles user interaction and menu commands  
- `objecttier.py` – Builds Python objects from database query results  
- `datatier.py` – Executes SQL queries and database actions  

## Notes
This project was created for learning purposes and demonstrates database access, SQL querying, and basic application structure using Python.
